# Santiago Weather App - Frontend MVP

Este es el proyecto final del Módulo 2. Es una aplicación de clima enfocada en 10 comunas de Santiago de Chile.

## Características
- Estructura HTML5 semántica.
- Diseño responsivo con Bootstrap 4.
- Interactividad con jQuery para visualización de detalles y pronóstico.

## Tecnologías utilizadas
- HTML5 / CSS3
- Bootstrap 4 (vía CDN)
- jQuery (vía CDN)
